### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/7akoom/carify.git
   ```
2. Install composer packages
   ```sh
   composer install
   ```
3. Generate App key
    ```sh
   php artisan key:generate
   ```

3. Migrate database and seeder
    ```sh
   php artisan migrate --seed
   ```

3. Start project
    ```sh
   php artisan serve
   ```
